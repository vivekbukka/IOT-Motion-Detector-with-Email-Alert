# IOT-Motion-Detector-with-Email-Alert
IoT Motion Detector with Email Alert is a smart security project using ESP32/ESP8266 and a PIR motion sensor. It detects movement in real time and instantly sends email alerts to the user through Wi-Fi. It can also trigger buzzer or lights for security. This project demonstrates IoT, automation, and remote monitoring concepts for homes and offices.
